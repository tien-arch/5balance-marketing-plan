## 🛠️ QUY TRÌNH "XỬ LÝ - MOVE"
1. Hoàn thiện file trong các folder `sprint/`.
2. Kiểm tra chất lượng (Audit) xem đã sát với GTM chưa.
3. Khi OK, chúng ta sẽ **Move** các file này về thư mục chung:
   - Các file chiến lược -> `/strategy`
   - Các bài viết/chiến thuật cụ thể -> `/tactics`
   - Dữ liệu nghiên cứu -> `/research`
